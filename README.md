## How to Use

1. Open `index.html` in a web browser
2. Fill in the form:
   - **Participant Name** - Full name of the participant
   - **Event Name** - Name of the event
   - **Date** - Date of the event
3. Click "Generate Certificate" to preview
4. Click "Download PDF" to save the certificate

## Certificate Template

The certificate includes:
- AI Innovation Society logo
- Department of Artificial Intelligence (STMU) logo
- "CERTIFICATE OF PARTICIPATION" header
- Participant name (bold)
- Event details with date
- Three signatories:
  - Dr. Hufsa Mohsin (HOD, Department of AI)
  - Sudais Khalid (President, AI Innovation Society)
  - Muhammad Tahir (Head, AI Innovation Society)
